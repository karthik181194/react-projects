## Run it on your machine

1. Clone the project

   ```sh
   git clone https://github.com/karthik181194/react-projects.git
   ```

1. Install the dependencies seperately

   ```sh
   npm install
   ```

3. Start the development server

   ```sh
   npm start
   ```
